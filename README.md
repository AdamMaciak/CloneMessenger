# CloneMessenger

Android app created as project to pass classes. 

## requirements
`Android Studio` and `Firebase`

## functionalities
* creating chats
* adding new contact
* adding new contacts to chat
* amount of unseen messages
* changing themes, username, user photo
* online/offline status in contacts
